FGO Dex Api Project

Welcome to a FGO Data single-page application, built with:

  • ASP.Net Core 3.1 for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

You'll find, for now, a Servant List on top of the page that will display every servant that is stored in the database. If you click on any of them it will display its ascensions if they are stored too.
